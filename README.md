## 技术选型

### 后端

* 基础框架：Spring Boot

* 持久层框架：Mybatis

* 安全框架：Shiro

* 模板引擎：Thymeleaf

* 其他：Druid、PageHelper

### 前端

* 基础框架：ElementUI（待定）

### 开发环境

* 语言： JDK1.8

* IDE： IDEA 2018.3

* 依赖管理： Maven

* 数据库： Mysql
